The main objective of this lab exercise is to generate a Payroll report for an employee in a company. 
The employee can work in 3 different job positions: (1) Programmer, (2) Sales, or (3) Office Worker.
This Application utilizes javax.swing and java.text libraries to request the employer data and display the resulting report.
The application contains 2 classes: (1) Payroll and (2) PayrollDemo
The folder contains the project files and an executable .JAR file.
