The objective of this lab exercise was to generate a GUI for the previous application developed in Lab5.
The main libraries utilized in this project were javax.swing, java.awt, and java.awt.event.
This project contains 3 classes: (1) MyApp which conains the main method for the application, (2) MyGui which contains the methods that generate the graphical interface, and (3) Payroll that contains the calculation methods of Lab5.
The folder contains the project files and a .JAR executable file.
